# Полезные команды
***
##  Команды для миграции
### 1. Создаем миграцию:
```alembic revision --autogenerate -m "Будет в названии миграции"```
### 2. Приводим БД к последней миграции:
```alembic upgrade head```
##  Команды docker'a
### 1. Запуск контейнера:
```docker compose up -d```
### 2 Остановка контейнера:
```docker compose down```
### 3. Просмотр запущенных контейнеров:
```docker ps -a```
